# Tema 10 - APIs and Services

Este tema trata sobre **APIs (Application Programming Interfaces)** y servicios, explicando los conceptos clave, los tipos de APIs y cómo consumir y crear servicios.

---

## Objetivos

* Entender qué es una API y un servicio.
* Diferenciar entre **REST**, **SOAP** y **GraphQL**.
* Aprender a consumir APIs externas.
* Crear una API simple propia.
