# Ejercicio 3 - Crear API propia

**Objetivo:** Practicar la creación de una API REST simple.

**Enunciado:**
1. Crea una API que gestione una lista de usuarios.
2. Implementa endpoints para:
- Obtener todos los usuarios (`GET /users`).
- Añadir un usuario (`POST /users`).
3. Prueba la API con Postman o cURL.
